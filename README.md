# TuNegocio

Plataforma para crear **landing pages simples** para pequeños negocios/freelancers, enfocada en **publicar rápido** y **cobrar online**.

## Estructura

- `Frontend/`: App en **Next.js** (landing builder + páginas públicas).
- `Backend/`: API en **ASP.NET Core 8** (ver su `Backend/README.md`).

## Ejecutar (Frontend)

```bash
cd Frontend
npm install
npm run dev
```

## Variables de entorno (Frontend)

Usa `Frontend/.env.example` como referencia y crea tu `.env` (no se commitea).
